
## Полное описание

```
FinCalendar — удобное приложение для ведения личного бюджета.

Основные возможности:
• Учёт доходов и расходов по категориям
• Несколько счетов (карты, наличные, накопления)
• Календарь с отметками по дням — видно когда и сколько потрачено
• Управление долгами и плановыми платежами
• Переводы между счетами
• Статистика и графики расходов
• Виджеты для рабочего стола
• Экспорт данных в JSON
• Полностью офлайн — все данные хранятся только на устройстве
• Поддержка русского и английского языков

Приложение не собирает и не передаёт ваши финансовые данные — 
всё хранится локально на вашем устройстве.
```
